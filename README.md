# 🚀 DigiTools Platform

---

## 📌 Project Overview

**DigiTools Platform** is a modern and user-friendly web application that brings together a collection of essential digital tools in one place.
It focuses on **simplicity, performance, and productivity**, helping users manage tools efficiently with a clean and responsive interface.

---

## 🧠 Key Idea

> ✨ “All your digital tools — organized, accessible, and easy to manage.”

## ✨ Features

### 🛒 Smart Cart System

- Add products/tools to cart easily
- Clean **1-column cart layout**
- Each item includes:
  - 📌 Name
  - 🖼️ Icon
  - 💰 Price
  - ❌ Remove button

---

### 🔢 Live Navbar Counter

- Shows total selected products
- Updates instantly when items are added/removed
- Improves user awareness and UX

---

### 💵 Total Price Calculation

- Automatically calculates total price of all selected items
- Updates dynamically with cart changes

---

### ❌ Remove Functionality

- Remove any specific product from cart
- Instant UI update
- Toast notification for feedback

---

### 🧹 Proceed to Checkout

- Clears all cart items with one click
- Acts as a reset system
- Confirmation via toast notification

---

### 🔔 Toast Notification System

- 📥 Add to Cart → Success alert
- ❌ Remove Item → Warning alert
- ✅ Checkout → Completion alert

## ⚡ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/mdshantosharker/DigiTools-Platform.git

# Navigate to project folder
cd DigiTools-Platform

# Install dependencies
npm install

# Run the project
npm run dev
```
